# AI Work Notes

This folder stores concise project-local AI planning and session evidence that
should be readable by Codex, Claude, and future assistants.

- `plans/`: specs, implementation plans, handoff briefs, review plans, QA plans,
  and release plans.
- `session-logs/`: dated logs of meaningful sessions, including what changed,
  what was verified, and what should be promoted to the shared knowledge store.

Keep these files in the project so Graphify can index them. Do not paste full
chat transcripts or full tool logs here unless the raw evidence itself is the
artifact being preserved. Promote reusable knowledge to
`/Users/kangsungbae/Documents/지식저장소`.
