# 스티커떼기

`스티커떼기`는 혼자 하는 즉시 만족형 모바일 캐주얼 게임 프로젝트다.

플레이어는 화면에 붙은 스티커의 모서리를 잡고 천천히 떼어낸다. 좋은 각도와 속도로 당기면 부드럽게 떨어지고, 무리하게 당기면 찢어지거나 접착 자국이 남는다. 핵심 재미는 "찢지 말고, 남기지 말고, 가장 깔끔하게 떼어내는 손맛"이다.
